# Flutter Learning Journey - Pizza App

A comprehensive Flutter learning project building a pizza ordering application from scratch.

## Tech Stack

### Core
- **Flutter SDK** (3.1.0+) - Cross-platform UI framework
- **Dart** - Programming language

### State Management & Architecture
- **Equatable** (^2.0.5) - Value equality for cleaner state management
- **RxDart** (^0.27.7) - Reactive programming with streams

### Backend Services
- **Firebase Authentication** (^4.16.0) - User authentication & authorization
- **Cloud Firestore** (^4.9.2) - NoSQL cloud database for real-time data
- **Firebase Storage** (^11.6.0) - Cloud storage for images (pizza photos)

### Utilities
- **Path** (^1.8.3) - File path manipulation

## Learning Goals
- Clean architecture and repository pattern
- Firebase integration and real-time data handling
- State management techniques
- CRUD operations with Firestore
- Image upload and retrieval
- Authentication flows
